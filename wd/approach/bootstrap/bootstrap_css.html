Layout

Definning a layout and positioning for a complex page can be very time consuming. Bootstrap layout offers a fixed-width layout with <div class="container"> and a class container-fluid for a fluid design where the width is automatically adjusted to fit the browser width.

Exercise

  • Copy the following template HTML code and save it as, layout.html.
  • <!DOCTYPE html>
<html lang="en">
   <head>
      <meta charset="utf-8">
      <title>Bootstrap Template</title>
      <link href="css/bootstrap.css" rel="stylesheet"> <!-- Bootstrap CSS -->
      <link href="css/custom.css" rel="stylesheet"> <!-- User defined CSS -->
   </head>      
   <body>
   
      <div class="container">
      
         <h1>Hello World Wide Web!</h1>
         
      </div>
               
      <script src="js/jquery-latest.js"></script> <!-- JQuery JS -->
      <script src="js/bootstrap.js"></script> <!-- Bootstrap JS -->
      
   </body>
</html>
  • Open layout.html in Chrome.
  • View the HTML and the CSS code in the developer window
    • Right-mouse click on the browser display and select "Inspect Element".
    • In the developer window, select the Elements tab and click on the Styles panel.
    • Scroll down the Styles panel and note how CSS is cumulatively applied starting with the root html element, followed by body and then the remainning elements in the HTML tree structure.
    • Click on the <div class="container"> element and note the CSS property width: 940px and margin-left and margin-right are set to auto to center the div.

Grid

For more complex layouts, Bootstrap offers a 12 column grid implemented with the row (and row-fluid) class with each row being defined by columns implemented with the span1 to span12 classes. Furthermore, row can be nested within other rows or columns and there is also an offset class to shift a column.

<div class="row">
   <footer>
      <div class="span12">

   </footer>

Typography

Typography, they say is over 90% of your design! At some point, watch this video.

Default typography can be very different accross browsers. Bootstrap typography uses the element type selctor to set a default typography for all elements, hence providing the same look & feel accross browsers.

Exercise

  • Copy the following template HTML code and save it as, typography.html.

  • View the HTML and the CSS code in the developer window
    • Right-mouse click on the browser display and select "Inspect Element".

    • Scroll down the Styles panel and note how CSS is cumulatively applied starting with the root html element, followed by body and then the remainning elements in the HTML tree structure.

Navigation

Bootstrap has a rich set of navigation components. All nav components, tabs, pills, and lists—share the same base markup and styles through the .nav class. Additionally you can then add classes, .nav-tabs, .nav-pills and so on.

Exercise

  • Copy the following template HTML code and save it as, navigation.html.
  • <!DOCTYPE html>
<html lang="en">
   <head>
      <meta charset="utf-8">
      <title>Bootstrap Template</title>
      <link href="css/bootstrap.css" rel="stylesheet"> <!-- Bootstrap CSS -->
      <link href="css/custom.css" rel="stylesheet"> <!-- User defined CSS -->
   </head>      
   <body>
   
      <div class="container">
      
         <nav>
            <ul class="nav nav-pills">
               <li class="active"><a href="#layout">Layout</a></li>
               <li><a href="#typography">Typography</a></li>
               <li><a href="#navigation">Navigation</a></li>
            </ul>
         </nav>
         
      </div>
               
      <script src="js/jquery-latest.js"></script> <!-- JQuery JS -->
      <script src="js/bootstrap.js"></script> <!-- Bootstrap JS -->
      
   </body>
</html>
  • Open navigation.html in Chrome.
  • View the HTML and the CSS code in the developer window
    • Right-mouse click on the browser display and select "Inspect Element".
    • In the developer window, select the Elements tab and click on the Styles panel.
    • Scroll down the Styles panel and note how CSS is cumulatively applied starting with the root html element, followed by body and then the remainning elements in the HTML tree structure.
    • Click on the <ul class="nav nav-pills"> element and note the CSS properties for the nav and nav-pills class selectors.
