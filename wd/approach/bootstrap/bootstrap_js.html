Bootstrap JS

Whereas in the past, DHTML markup such as onclick="do_js();" was used, now with more consistant support in browsers unobtrusive Javascript can be used to keep events seperate from the HTML code by using selectors rather than DHTML to trigger events and HTML5 data- and aria- attributes to pass any necessary data to Javascript.

Seperation of content (XHTML5) from presentation (CSS3) from events (unobtrusive JS) is key to modern web development.

Fo a complete list of JS components, see the Bootstrap Javascript documentation. As an illustration, consider the following example; click on the modal button to see the result before considering the code in the exercise.

Exercise

  • Copy the following template HTML code and save it as, modal.html.
  • <!DOCTYPE html>
<html lang="en">
   <head>
      <meta charset="utf-8">
      <title>Bootstrap Template</title>
      <link href="css/bootstrap.css" rel="stylesheet"> <!-- Bootstrap CSS -->
      <link href="css/custom.css" rel="stylesheet"> <!-- User defined CSS -->
   </head>      
   <body>
   
      <div class="container">
      
         <h1>Bootsrtap unobtrusive JS</h1>
            <!-- Modal -->
            <!-- Button to trigger modal -->
            <p><a href="#myModal" role="button" class="btn" data-toggle="modal">Launch modal demo</a></p>

            <!-- modal panel -->
            <div id="myModal" class="modal hide fade" tabindex="-1" 
               role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
               <div class="modal-header">
                  <button type="button" class="close" data-dismiss="modal" aria-hidden="true">×</button>
                  <h3 id="myModalLabel">Modal header</h3>
               </div>
               <div class="modal-body">
                  <p>One fine body…</p>
               </div>
               <div class="modal-footer">
                  <button class="btn" data-dismiss="modal" aria-hidden="true">Close</button>
                  <button class="btn btn-primary">Save changes</button>
               </div>
            </div>         
      </div>
               
      <script src="js/jquery-latest.js"></script> <!-- JQuery JS -->
      <script src="js/bootstrap.js"></script> <!-- Bootstrap JS -->
      
   </body>
</html>
  • Open modal.html in Chrome.
  • View the HTML and the CSS code in the developer window
    • Note the hyperlink href="#myModal" which is the id of the modal panel, <div id="myModal". This is an example of how the association between the hyperlink and the div is made unobtrusively without additional markup such as onclick
    • The HTML5 data- attributes such as data-toggle and data-dismiss provide the necessary data for the JS to complete the transition.
    • Accessibility is catered for with the aria- attributes; for example, note the aria-labelledby="myModalLabel" specifies the id for <h3 id="myModalLabel">Modal header</h3>. A screen reader can therefore read out the content of the heading.
    • The presentation as always is defined with the CSS classes; right-mouse click on the modal button and select "Inspect Element" to see the CSS properties.
    • In the developer window, select the <div id="myModal" and note the CSS has display: none;. The modal hide fade CSS classes determine the presentation of the modal panel.
