What is it all about ... ?

Our aim is an introduction to the Bootstrap framework developed by the good folks at Twitter for CSS and JS.

Where to start?

Using a framework is essentially through a set of selector names that are defined in the framework with associated CSS and JS.

A good start is to understand how selectors are used in practice. Test yourself (and if necessary revise) the answers to the questions on this page. Once the answers are clear in your mind, you are ready to start.

Bootstrap template

We have downloaded a copy of the CSS and JS source code from the Bootstrap website and saved it in the css and js directories. The following XHTML5 template uses these Bootstrap files.

<!DOCTYPE html>
<html lang="en">
   <head>
      <meta charset="utf-8">
      <title>Bootstrap Template</title>
      <link href="css/bootstrap.css" rel="stylesheet"> <!-- Bootstrap CSS -->
      <link href="css/custom.css" rel="stylesheet"> <!-- User defined CSS -->
   </head>      
   <body>
   
      <h1>Hello World Wide Web!</h1>
               
      <script src="js/jquery-latest.js"></script> <!-- JQuery JS -->
      <script src="js/bootstrap.js"></script> <!-- Bootstrap JS -->
      
   </body>
</html>
  • Note the <link href="css/bootstrap.css" rel="stylesheet"> element in the head section linking to the Bootstrap CSS file allowing use of selectors defined in Bootstrap Base CSS, Scaffolding for layouts and Components for a variety of common CSS components.
  • The <link href="css/custom.css" rel="stylesheet"> is specified after the Bootstrap CSS link. This is allows for any additional custom CSS to be included in a seperate file.
  • The two <script> elements are included at the end of the file for faster page display and include JQuery and Bootstrap JS. This allows for reuse of Bootstrap JS components
  • We shall use the above code as a template in all the CSS and JS examples.
